# Changelog

## v2.1.0 - Nexus Intelligence Lab
*Released: 2025-12-08*

### New Features
- **Legacy News Hub -> Nexus Intelligence Lab**: Complete overhaul of the news interface.
- **Subject Radar**: Track keywords across all news sources with persistent counters.
- **Watch Tab**: Specialized video feed filtering with auto-detection for Banned.Video, InfoWars, and YouTube.
- **AI Research Analyst**: Integrated LLM analysis for research folders (Executive Summaries, Key Insights). "Synthesize Research" button added.
- **System Status**: New "System Online" and source count indicators.

### Improvements
- **Performance**: Lazy loaded `ResearchPanel` for faster initial page load.
- **UI/UX**: "Nexus Intelligence" gradient branding. Improved glassmorphism on Research Panel.
- **Developer Experience**: Removed debug console logs from Chat and Agents.

## v2.0.0 - Growth Phase Complete
*Released: 2025-12-08*

- **Foundation**: Complete architectural stability.
- **Installable**: Application packaged for local installation (LuxRig).
- **Voice Control**: Integrated web speech API for identifying user intent.
- **Advanced Agents**: Added Architect, QA, Security, and DevOps personas.
