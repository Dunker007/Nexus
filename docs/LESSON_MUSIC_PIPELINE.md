## Module 1: Songwriter Agents
**Duration:** 45 minutes

### Learning Objectives
- Use AI agents to collaborate on songwriting
- Generate structured prompts for Suno
- Understand genre/mood/style tags

### 1.1 The Songwriting Room
Multiple AI agents collaborate like a real songwriting team:

| Agent | Role |
|-------|------|
| **Lyricist** | Writes lyrics, hooks, emotional content |
| **Composer** | Style, arrangement, mood, instruments |
| **Critic** | Reviews and refines |
| **Producer** | Commercial viability, final polish |

### 1.2 Creating Effective Suno Prompts
Good prompts include:
- **Genre:** "upbeat pop", "dark synthwave", "acoustic folk"
- **Mood:** "energetic", "melancholic", "triumphant"  
- **Instruments:** "piano-driven", "heavy bass", "acoustic guitar"
- **Tempo:** "fast-paced", "slow ballad"

### 1.3 Exercise
1. Open Songwriting Room
2. Enter theme: "Song about chasing dreams"
3. Let agents debate â†’ Generate final prompt
4. Review the structured output

## Module 2: Suno AI Music Generation
**Duration:** 45 minutes

### Learning Objectives
- Generate complete tracks from prompts
- Download audio for video creation
- Manage your Suno library

### 2.1 Using Suno
```
1. Go to suno.ai
2. Paste your songwriter prompt
3. Generate 2-3 variations
4. Pick best track â†’ Download audio
```

### 2.2 Pro Tips
- Generate multiple variations, pick the best
- Use "Extend" for longer intros/outros
- Save your favorites to playlists
- Download as MP3 or WAV

### 2.3 Exercise
1. Take your songwriter output
2. Generate 2 variations in Suno
3. Listen and compare
4. Download the winner

## Module 3: Neural Frames Video Creation
**Duration:** 1 hour

### Learning Objectives
- Create audio-reactive music videos
- Use Autopilot mode for quick videos
- Export YouTube-ready 4K video

### 3.1 Neural Frames Basics
Neural Frames analyzes 8 audio stems to create perfectly synced visuals:
- Drums/percussion
- Bass
- Vocals
- Melody
- And more...

### 3.2 Two Modes

| Mode | Use When |
|------|----------|
| **Autopilot** | Quick video, minimal effort |
| **Custom** | Full creative control |

### 3.3 Autopilot Workflow
```
1. Upload your Suno audio
2. Choose visual style/theme
3. Click Generate
4. Wait ~5-10 minutes
5. Preview â†’ Export 4K MP4
```

### 3.4 Custom Workflow
- Frame-by-frame control
- Text prompts per section
- Lyric sync (auto or manual)
- Multiple visual styles

### 3.5 Exercise
1. Upload your Suno track
2. Try Autopilot first
3. Preview the result
4. Export as 4K MP4

---

## Module 4: YouTube Publishing
**Duration:** 30 minutes

### Learning Objectives
- Upload videos efficiently
- Optimize metadata for discovery
- Schedule releases

### 4.1 Upload Process
```
1. Go to YouTube Studio
2. Upload your Neural Frames MP4
3. Add title, description, tags
4. Set thumbnail
5. Choose public/scheduled
6. Publish!
```

### 4.2 Metadata Best Practices

| Element | Tips |
|---------|------|
| **Title** | Catchy, include genre keywords |
| **Description** | Full lyrics, credits, links |
| **Tags** | Genre, mood, similar artists |
| **Thumbnail** | Eye-catching, custom image |

### 4.3 Automation (Advanced)
Our platform can automate upload via YouTube API:
- Automatic metadata from songwriter agents
- Scheduled publishing
- Batch uploads

### 4.4 Exercise
1. Upload your finished video
2. Add complete metadata
3. Set as unlisted for review
4. Schedule or publish
